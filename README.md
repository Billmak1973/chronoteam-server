# ChronoTeam 專業二手名錶交易平台

## 項目簡介
ChronoTeam 是一個致力於提供安全、透明、專業的二奢名錶交易生態圈平台。系統涵蓋了完整的買賣家交易閉環、豐富的社區互動機制以及嚴格的帳號安全與處罰體系。

## 核心功能模塊

### 🛒 買家與商品中心
- **商品瀏覽與篩選**：支持多維度篩選（分類、品牌、價格區間）及多種排序方式。
- **商品詳情與收藏**：查看商品詳細參數與來源介紹，支持一鍵收藏/取消收藏。
- **購物車系統**：React 實現的購物車下拉菜單與獨立頁面，支持樂觀更新、全選/單選結算、數量調整與庫存校驗。
- **訂單與支付**：支持線上模擬支付與線下門店支付選擇；訂單狀態追蹤（待付款/已支付），支持訂單明細展開與取消/刪除訂單。
- **到貨通知**：缺貨商品一鍵訂閱，補貨後自動接收系統通知。
- **瀏覽歷史**：自動記錄瀏覽軌跡，支持按時間段清理。

### 💰 賣家與財務中心
- **發佈出售申請**：支持多圖上傳（錶面、機芯、保卡等）、附件勾選、成色評估，可選擇「平台直接買斷」或「平台代為寄售」模式。
- **寄售與出售管理**：追蹤申請進度（待鑑定、已報價、寄售中等），支持取消申請或徹底刪除記錄。
- **財務結算記錄**：自動計算並展示累計總成交額、平台服務費扣除及實際淨收益。

### 💬 社區互動與評論系統 (React 深度定制)
- **多層級評論**：支持根評論與樓中樓回覆，支持分頁與多種排序（最新、最早、最多點贊）。
- **點贊/踩與防刷機制**：互動操作綁定速率限制（Rate Limit），頻繁操作觸發自動封禁並發送系統通知。
- **評論管理**：作者可修改/刪除評論；管理員可置頂評論或刪除評論並填寫自定義原因。
- **舉報系統**：兩級分類選擇（違法、色情、網暴、詐騙等），防重複舉報攔截。

### 🛡️ 帳號安全與處罰體系
- **多維度安全設置**：支持原密碼或「安全問答」驗證修改密碼；帳戶安全等級評估；近期登錄設備與 IP 記錄。
- **禁言與拉黑機制**：
  - **普通用戶**：雙向禁言（互相無法回覆）。
  - **管理員**：全局禁言（設定時長）、永久拉黑。
- **申訴系統**：用戶可針對禁言、拉黑、評論刪除等處罰提交申訴，管理員審核後自動解除或維持處罰。
- **消息通知中心**：區分「系統通知」（處罰、刪除）與「互動消息」（回覆、@），導航欄實時紅點提示。

## 管理員後台功能

### ✅ 已實現
- **商品微調**：管理員可在商品詳情頁直接修改價格、庫存、分類、品牌及圖文詳情。
- **評論區執法**：一鍵置頂評論、刪除違規評論並選擇/填寫刪除原因（系統自動發送通知給用戶）。
- **用戶處罰與申訴審核**：執行全局禁言、永久拉黑操作；查看並審核用戶的申訴請求。
- **到貨通知管理**：查看缺貨商品的等待名單，一鍵批量發送補貨系統通知。
- **數據儀表板 (UI)**：管理後台基礎佈局與菜單框架已搭建。

## 技術棧
- **前端**：React (Hooks, 狀態管理, 樂觀更新), Thymeleaf, HTML5/CSS3, Vanilla JS, FontAwesome
- **後端**：Spring Boot, Spring Security, Spring Data JPA
- **數據庫與緩存**：MySQL, Redis
- **其他**：RESTful API, 文件存儲服務, 異步任務調度
